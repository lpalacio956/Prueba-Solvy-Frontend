## Description

Documentación del Proyecto

Backend - Nest.js con PostgreSQL y Prisma

1. Instalación y Configuración

Requisitos previos

Node.js (versión recomendada: LTS)

PostgreSQL instalado y en ejecución

Prisma CLI instalado globalmente (npm install -g prisma)

Pasos de instalación

Clonar el repositorio:

https://github.com/lpalacio956/Prueba-Solvy

Instalar dependencias:

npm install

Configurar el entorno:

Crear un archivo .env basado en .env.example con la configuración de la base de datos:

DATABASE_URL="postgresql://postgres:sachi@localhost:5432/nestdba?schema=public"

Configurar Prisma:

npx prisma migrate dev --name init
npx prisma generate

Ejecutar el servidor:

npm run start

2. Endpoints principales

Método

Endpoint

Descripción

GET

/productos

Obtener todos los productos

GET

/productos/:id

Obtener un producto por ID

POST

/productos

Crear un nuevo producto

PATCH

/productos/:id

Actualizar un producto

DELETE

/productos/:id

Eliminar un producto

Mediante swagger puede vizualisar los metodos del CRUD
